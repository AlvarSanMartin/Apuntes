#### GAIA-X
- Gaia-x es una alternativa al sistema de datos IDS
- Ecosistema para la confianza en la compartición de datos
- Como se alinea con el European Data Estrategy

 >A **Node** is a general computational resource or infrastructure element, which may range from an edge resource to a virtual machine, a container, a data centre or any other generic infrastructure building block that services can be deployed on. Nodes are placed in the GAIAX Infrastructure Ecosystem.

- Concepto de federación de servicios, conectan la infraestructura con los datos
- GAIA = infraestructura cloud
- IDS = soberanía de datos

El concepto de identidad de IDS gestiona certificados de manera dinámica sin  cancelarlos y reemitirlos, se pueden crear para dispositivos y servicios, importante la figura 10 del [pdf](./pdfs/IDSA-GAIA.pdf).
