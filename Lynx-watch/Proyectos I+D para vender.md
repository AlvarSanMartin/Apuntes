# Basados en i+d existentes:
### Ceindus: Software de predicción de picos de consumo y agrupacion de patrones
Tecnalia nos vende un software de clustering de consumos por patrón y franja horaria para planificar la potencia contratada y la producción.

Pros: 
- Ya tenemos contenedores para hacer una demo.
- Son algoritmos de clustering que podemos replicar.
- Infraestructura sencilla.
Cons: 
- Dudas sobre las predicciones
- Hay que pagar licencia Mysql.
- No tenemos propiedad del código, dependemos en parte de tecnalia.
- Actualizaciones de datos diarias.

### Aztertuz: Visualización de BIM con datos en tiempo real + GMAO 
Visor-editor BIM con un servicio de datos en tiempo real para visualizar la alarmas de Lynx y datos en tiempo real del modulo de Watch.

Pros:
- Trabajamos con Imanol
- Tenemos propiedad de los resultados
- Se hace 100% para nosotros
Cons:
- No podemos editar el código del visor.


### I+D de vicomtech: toma de datos de alta frecuencia
Nos desarrollan un modulo de toma de datos y tratamiento en tiempo real.

Pros: 
- Propiedad de los resultados.
- Encaja con Lynx
Cons: 
- No da para un producto per-se.
- 2 años para una entrega.
