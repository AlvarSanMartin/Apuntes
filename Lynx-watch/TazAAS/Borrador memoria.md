## PT5 - p44
> Resultados esperados

Como resultado del desarrollo del proyecto TRAZAAS Dominion espera mejorar sus métodos de extracción, almacenamiento y compartición de datos en entonos industriales y de producción eléctrica. Se espera mejorar la frecuencia de recogida de datos, su tratamiento para poder almacenar un mayor volumen y estandarizar su compartición de acuerdo a estándares internacionales como se propone con IDSA, Gaia-X y Catena-X.
- Mejora de la captura de datos: El objetivo es mejorar la frecuencia, latencia y sistematizar la captura de datos en aquellos lugares que una mayor resolución o velocidad de respuesta sea necesaria. 
- Generación de AAS de tipo2: Se pretende establecer estandartes de comunicación entra los dispositivos físicos y el gemelo digital, se espera mejorar los procesos actuales para establecer comunicaciones fiables en tiempo real bajo el estándar AAS.
- Simulaciones sobre gemelos digitales: Para poder hacer estimaciones sobre el impacto, principalmente en gasto energético, producción e impacto ambiental de modificar los parámetros de producción sobre un conjunto de activos.
- Testeo y validación de gemelos digitales: Permitiendo conocer con que grado de precisión se están extrayendo los datos de cada activo, con la finalidad de conocer el estado de los sensores y su fiabilidad.
- Espacio de datos: Se espera complementar el sistema de almacenaje de datos desarrollado en Dominion con un servicio federado para permitir la compartición de datos de forma que sean compatibles con el estándar AAS y los involucrados en ADE. Se espera desarrollar una API  que permita el acceso a datos de las AAS desplegadas de forma que se puedan crear servicios nuevos de forma independiente que puedan consumir los datos.

Gracias a que el proyecto esta enfocado en la adopción de estándares internacionales podemos esperar mas facilidades a la hora de captar clientes fuera de España.
